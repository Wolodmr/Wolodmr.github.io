Job-Market Monitor

Telegram + Jooble API · Python

Context

Filtering relevant analyst roles from job channels is noisy and time-consuming. I built a small monitor to triage posts automatically.

Method

  • Sources: Telegram channel history + Jooble API.
  • Keyword filters (e.g., “data analyst”, “SQL”, “Belgrade”, “remote”).
  • Deduping, simple scoring, and alerts to a private chat.

Results

  • Saved ~2 hours/week by automating search + triage.
  • Alert precision of ~20% (“% of alerts that were genuinely relevant/applicable”) after tuning keywords.

Representative snippet

# pseudo-code
for post in stream():
    text = normalize(post.text)
    score = score_text(text, POSITIVE_KEYWORDS, NEGATIVE_KEYWORDS)
    if score >= THRESHOLD:
        send_alert(post.link, summary(text))
